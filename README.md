# Braccio_Lemniscate

You can find the Onshape 3D model of the robot [here](https://cad.onshape.com/documents/b1e6a25ed68ab1d075ca2128/w/ffa62a9eea9683b2658504c0/e/ca61c915463be64d20479233)
